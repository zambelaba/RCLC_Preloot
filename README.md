"# RCLC_Preloot" 

Pour l'import du ratio dans LootCouncil : 

Prérequis : 
Telecharger cet addon : https://github.com/zambelaba/RCLC_Preloot/archive/refs/tags/0.1.1.zip
Installer RCLootCouncil

Prépa raid :
- Sur le gsheet Preloot -> générer les ratio préloot en cliquant sur le logo engrenage
- Sur le gsheet Preloot -> copier la chaine de caractère encodée qui est générée en cellule S7
- En jeu -> /rcpl
- En jeu -> Coller le chaine de caractere encodée
- En jeu -> Click sur "Set Ratio"

Test de l'affichage des ratios importés dans lootcouncil:
- /rclc test
